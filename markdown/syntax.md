# Novalat 语法总览（表格整合版）

## 一、词根与派生

| 项目         | 规则说明                                                                 |
|--------------|--------------------------------------------------------------------------|
| 词根提取     | 取名词第一格，删去最后一个音节的元音及其之后部分，剩余为词根，性别继承自该名词 |
| 派生示例     | ama（名）→ am（根）→ amāre（动，阴）、amno（形）、amü（副）、amduis（动名） |

---

## 二、词性后缀

| 词性   | 后缀                     |
|--------|--------------------------|
| 代词   | -i（如 mi, ti, vi）      |
| 形容词 | -no                      |
| 名词   | 见名词变格            |
| 副词   | -ü / -li             |
| 连词   | -loc                 |
| 助词   | -h（不发音）             |

## 五、动词系统（大表格）

### 5.1 动词种类与不定式

| 类型       | 词根来源   | 语义特征           | 不定式尾 | 示例         |
|------------|------------|--------------------|----------|--------------|
| 阴性动词   | 阴性词根   | 一般动作           | -āre     | amāre（爱）  |
| 阳性动词   | 阳性词根   | 一般动作           | -ēre     | fortitudēre（勇敢） |
| 第一中性   | 中性词根   | 状态（睡、在、是） | -ere     | estere（是） |
| 第二中性   | 中性词根   | 被动行为/主动态    | -ere     | monere（劝告） |

### 5.2 变位要素

| 要素         | 形式                                       |
|--------------|--------------------------------------------|
| 词干         | 词根 + 不定式尾元音（a/e/e/e）            |
| 时态中缀     | 现在：无；将来：-bi-；过去：-ba-；现在完成：无；将来完成：-ri-；过去完成：-ra- |
| 人称词尾（主动，现在类） | -m, -s, -t, -mus, -tis, -nt               |
| 人称词尾（主动，完成类） | -mī, -vīs, -dīt, -mīmus, -vītis, -dīnt    |
| 人称词尾（被动，通用）   | -r, -ris, -tur, -mur, -ritis, -ntur       |

### 5.3 变位总表（以 am- 为例）

| 时态       | 主动态单数 (1/2/3)        | 主动态复数 (1/2/3)        | 被动态单数 (1/2/3)  | 被动态复数 (1/2/3)    |
|------------|----------------------------|----------------------------|----------------------|------------------------|
| 现在       | amām, amās, amāt           | amāmus, amātis, amānt      | amār, amāris, amātur | amāmur, amāritis, amāntur |
| 将来       | amābim, amābis, amābit     | amābimus, amābitis, amābint| amābir, amābiris, amābitur | amābimur, amābitis, amābintur |
| 过去       | amābam, amābas, amābat     | amābamus, amābatis, amābant| amābar, amābaris, amābatur | amābamur, amābaminī, amābantur |
| 现在完成   | amāmī, amāvīs, amādīt      | amāmīmus, amāvītis, amādīnt| （用助动词+过去分词） |                          |
| 将来完成   | amārīm, amārīs, amāridīt   | amārīmus, amārītis, amāridīnt | （同上）           |                          |
| 过去完成   | amārām, amārās, amāradīt   | amārāmus, amārātis, amāradīnt | （同上）           |                          |

## 六、动名词与分词

| 类别       | 构成                | 示例（am-） | 说明/变格                      |
|------------|---------------------|-------------|--------------------------------|
| 动名词     | 词干 + -duis        | amāduis      | 按第三变格法（中性）变格       |
| 现在分词（主动） | 词干 + -nt          | amānt       | 形容词性，变格同形容词         |
| 过去分词（主动） | 词干 + -nd          | amānd       | 同上                           |
| 将来分词（主动） | 词干 + -nil         | amānil      | 同上                           |
| 现在分词（被动） | 词干 + -rint        | amārint     | 同上                           |
| 过去分词（被动） | 词干 + -rand        | amārand     | 同上                           |
| 将来分词（被动） | 词干 + -ril         | amāril      | 同上                           |

## 八、句法

- **语序**：主语、宾语、谓语可任意排列
- **主语省略**：动词人称已含主语信息，代词可省略（除非强调）
- **修饰成分**：定语、状语、补语、表语自由排列

---

## 九、语气

| 语气类型     | 构成方式                                  | 示例                            |
|--------------|-------------------------------------------|---------------------------------|
| 命令语气     | 动词不定式                                | Amāre!（去爱！）                |
| 特殊疑问     | cūit- + 词根 置句首                       | Cūittemp venbīs?（你何时来？）  |
| 一般疑问     | 主题后加 -net                             | Frater estenet?（是兄弟吗？）    |
| 肯定反问     | 主题后加 -nam                             | Estnam?（难道不是？）           |
| 否定反问     | 主题后加 -none                            | Estnone?（难道是？）            |
| 感叹         | cūitü + 主题                              | Ve cūitü amām!（我多么爱你！）   |
| 虚拟         | siloc + 主题                              | Siloc ve amām.（如果我爱你）   |
